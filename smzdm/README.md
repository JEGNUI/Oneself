## 配置

```properties
[MITM]
*.smzdm.com

[rewrite_local]
# 189及以前版本
^https:\/\/www\.smzdm\.com\/?.? url script-response-body smzdm.cookie.js
# 190及以后版本
^https:\/\/www\.smzdm\.com\/?.? url script-request-header smzdm.cookie.js

[task_local]
1 0 * * * smzdm.js
```
